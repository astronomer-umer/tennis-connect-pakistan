## 📱 PWA - Install on Any Device

The app is a **Progressive Web App (PWA)** that works like a native app:

| Platform | How to Install |
|----------|----------------|
| **iPhone/iPad** | Safari → Share → Add to Home Screen |
| **Android** | Chrome → Menu → Install App |
| **Windows** | Edge → Menu → Apps → Install |
| **Mac** | Safari → File → Add to Dock |
| **Linux** | Chrome → Menu → More Tools → Create Shortcut |

### ✅ PWA Features
- 📲 Install like native app
- 🔄 Auto-updates
- 📴 Works offline
- 🔔 Push notifications ready
- ⚡ Lightning fast

## 🏪 Publish to App Stores (as PWA)

You can publish the PWA directly to all stores using [PWABuilder](https://www.pwabuilder.com):

```bash
# 1. Go to pwabuilder.com
# 2. Enter: https://tennis-connect-pakistan.vercel.app
# 3. Download packages for each store
# 4. Submit to respective stores
```

### Microsoft Store (Windows)

1. Visit [PWABuilder](https://www.pwabuilder.com)
2. Enter your deployed URL
3. Click **Windows** → **Store Package**
4. Download the `.msix` package
5. Submit to [Microsoft Partner Center](https://partner.microsoft.com)

**Store Details:**
- Name: **Tennis Connect Pakistan**
- Category: Sports
- Age Rating: 3+
- Price: Free
- Region: Pakistan (expand globally)

### Google Play Store (Android)

1. Visit [PWABuilder](https://www.pwabuilder.com)
2. Enter your deployed URL
3. Click **Android** → **Play Store Package**
4. Download the `.aab` (App Bundle)
5. Sign up for [Google Play Console](https://play.google.com/console)
6. Create App → Upload `.aab`

**Play Store Assets Needed:**
- App Icon: 512x512 PNG
- Feature Graphic: 1024x500 PNG
- Screenshots: Phone (min 320px wide)

### Apple App Store (iOS/macOS)

1. Visit [PWABuilder](https://www.pwabuilder.com)
2. Enter your deployed URL
3. Click **iOS** → **App Store Package**
4. Download the package
5. Set up [Apple Developer Account](https://developer.apple.com) ($99/year)
6. Use Xcode or Transporter to upload

**App Store Assets Needed:**
- App Icon: 1024x1024 PNG
- Screenshots: iPhone 6.7", 6.5", 5.5"
- App Preview: Optional video (30s max)

### Amazon Appstore (Fire Tablets)

1. Visit [PWABuilder](https://www.pwabuilder.com)
2. Enter your deployed URL
3. Click **Amazon** → **Amazon Package**
4. Submit to [Amazon Developer Console](https://developer.amazon.com)

## 🚀 Deploy Your Own

### One-Click Deploy

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/your-username/tennis-connect-pakistan)

### Manual Deploy

```bash
# Clone
git clone https://github.com/your-username/tennis-connect-pakistan.git
cd tennis-connect-pakistan

# Install
npm install

# Environment Variables
cp .env.example .env.local
# Edit .env.local with your values

# Develop
npm run dev

# Build for production
npm run build

# Deploy to Vercel
npx vercel --prod
```

### Required Environment Variables

```env
# Turso Database (SQLite in the cloud)
TURSO_DATABASE_URL=libsql://your-db.turso.io
TURSO_AUTH_TOKEN=your-token

# Admin Panel Password
ADMIN_PASSWORD=your-secure-password

# Better Auth
BETTER_AUTH_SECRET=generate-32-char-secret
BETTER_AUTH_URL=https://your-domain.vercel.app
```
